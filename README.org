* Itineraries

** CUBA1 (Western Route)
Habana → Viñales → Soroa → Playa Larga → Cienfuegos → Trinidad

*Note:* María la Gorda removed per Marcel's request. No loop back to Habana.

** CUBA2 (West to East Cross-Country)
Main route:
- Habana → Viñales → Soroa → Playa Larga → Cienfuegos → Trinidad
- Trinidad → Camagüey → Santiago de Cuba → Baracoa
- Baracoa ↔ Santiago de Cuba (bidirectional)
- Santiago de Cuba → Habana (straight nightbus line)

Optional eastern loop (anti-clockwise):
- Baracoa → Faro de Maisi → Yumurí → Baracoa
